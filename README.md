# FIMS
# Структура проекта

## frontend/

```plaintext
frontend/
├── src/
│   ├── assets/             # Для статических файлов (изображения, иконки, стили)
│   ├── components/         # Общие компоненты приложения
│   │   ├── auth/           # Компоненты для авторизации и логина
│   │   │   ├── Login.tsx   # Вход в систему
│   │   │   └── Logout.tsx  # Выход из системы
│   │   ├── admin/          # Компоненты панели администратора
│   │   │   └── AdminPanel.tsx
│   │   ├── stock/          # Компоненты склада
│   │   │   └── StockWorkerPanel.tsx
│   │   ├── cashier/        # Компоненты кассира
│   │   │   └── CashierPanel.tsx
│   │   └── shared/         # Переиспользуемые компоненты (например, таблицы или модальные окна)
│   │       ├── Navbar.tsx
│   │       └── Footer.tsx
│   ├── context/            # Контексты для управления состоянием через React Context API
│   │   └── AuthContext.tsx # Контекст для авторизации
│   ├── hooks/              # Пользовательские хуки
│   │   └── useFetch.ts     # Хук для упрощения работы с API
│   ├── pages/              # Страницы приложения для маршрутизации
│   │   ├── AdminPage.tsx   # Главная страница для администратора
│   │   ├── StockPage.tsx   # Главная страница для складского работника
│   │   ├── CashierPage.tsx # Главная страница для кассира
│   │   └── NotFound.tsx    # Ошибка 404
│   ├── routes/             # Настройка маршрутов внутри приложения
│   │   └── AppRoutes.tsx   # Основной маршрутный файл
│   ├── services/           # API-запросы или взаимодействие с сервером
│   │   ├── authService.ts  # Обработка авторизации
│   │   ├── productService.ts # Запросы, связанные с товарами
│   │   └── userService.ts  # Запросы, связанные с пользователями
│   ├── App.css
│   ├── App.tsx
│   ├── index.css
│   ├── main.tsx
│   └── vite-env.d.ts
